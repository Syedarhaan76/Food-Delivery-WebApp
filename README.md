# Food-Delivery-WebApp

A fully responsive Food Delivery Website built using HTML, CSS, and JavaScript.
This project showcases a modern UI for browsing food items, categories, offers, and a clean ordering interface.
Deployed and hosted on Netlify for blazing-fast performance.

🚀 Live Demo

https://shahi-kashmir.netlify.app/

⭐ Features

# Fully responsive design

# Clean and modern UI

# Smooth animations and transitions

# Deployed on Netlify


🛠 Tech Stack
Technology	Usage
HTML5	Structure of the website
CSS3	Styling, layout, responsiveness
JavaScript (ES6+)	UI interactions, cart logic
Netlify	Hosting & deployment
